# FIGHTERS-ARENA
Fighter's Arena é um jogo RPG feito em Java

Funcionalidades:

  Tela de login:
  - Login
  - Cadastrar Usuário
  - Sair

  Menu principal:
  - Gerenciar personagens
  - Iniciar batalha
  - Loja
  - Logout

  Gerenciar personagens:
  - Criar novo personagem
  - Listar personagens
  - Selecionar personagens
  - Voltar

  Iniciar batalha:
  - PvE (vs CPU)
  - PvP (Local)
  - Voltar

  Loja:
  - Habilidade 1
  - [...]
  - Voltar
